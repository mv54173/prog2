# Rukomet — pobjednici prvog mjeseca

Deklarirati strukturu `rukomet` u koju će se spremiti ovakvi ulazni podaci:
naziv prve momčadi, naziv druge momčadi, `golovi_1`, `golovi_2`, `datum`.
Nazivi momčadi se mogu sastojati od više riječi, dok varijabla `golovi_1`
predstavlja broj golova koje je zabila prva momčad, a varijabla `golovi_2`
predstavlja broj golova koje je zabila druga momčad i može biti samo cijeli broj.

U program se prvo unosi broj utakmica `n` (`0 < n < 100`).

U datoteku `pobjednici.txt` zapisati u jednom retku (s razmakom) imena svih
pobjednika u prvom mjesecu.

## Primjer ulaznih podataka

```
2
Njemacka
Skotska
21
23
14.04.2024.
Spanjolska
Hrvatska
18
25
15.01.2024.
```

## Očekivani izlaz (`pobjednici.txt`)

```
Hrvatska
```
