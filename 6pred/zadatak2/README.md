# Zadatak 2 – Duplo prepisivanje redaka

Napisati program koji iz datoteke `pjesme.txt` učitava retke i svaki učitani redak dopisuje na kraj datoteke `dnevnik.txt`. Svaki redak mora se pojaviti dva puta u datoteci `dnevnik.txt`.

Koristiti `fgets` u `while` petlji i `fputs` za pisanje.

## Primjer

**pjesme.txt:**
```
Sunce sja nad poljem zlatnim
Vjetar nosi pjesmu daleku
Ptice pjevaju u krosnjama
```

**dnevnik.txt** (nakon 1. pokretanja):
```
Sunce sja nad poljem zlatnim
Sunce sja nad poljem zlatnim
Vjetar nosi pjesmu daleku
Vjetar nosi pjesmu daleku
Ptice pjevaju u krosnjama
Ptice pjevaju u krosnjama
```
