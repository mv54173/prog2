# Zadatak 1 – Retci koji počinju velikim slovom

Napisati program koji iz datoteke `recenice.txt` učita sve retke (maksimalno 200 znakova po retku) i na ekran ispiše samo one retke koji počinju velikim slovom.
Koristiti `fgets` u `while` petlji do `NULL`.

## Primjer

**recenice.txt:**
```
Ovo je recenica koja pocinje velikim slovom.
ovo je recenica koja pocinje malim slovom.
Programiranje je zabavno.
jezik C je mocan alat.
Datoteke su korisne za pohranu podataka.
```

**Ispis na ekran:**
```
Ovo je recenica koja pocinje velikim slovom.
Programiranje je zabavno.
Datoteke su korisne za pohranu podataka.
```
