# Zadatak 3 – Statistika brojeva iz datoteke

U datoteci `brojevi.txt` nalazi se neodređen broj cijelih brojeva odvojenih razmakom ili novim redom. Napisati program koji ih sve pročita pomoću `fscanf` i ispiše njihov broj, zbroj i prosjek.

Funkcija `prebrojiIZbroji` vraća broj brojeva kao povratnu vrijednost, a zbroj vraća preko pokazivača.

## Primjer

**brojevi.txt:**
```
10 20 30
40 50
60 70 80 90 1
```

**Ispis:**
```
10 451 45.10
```
